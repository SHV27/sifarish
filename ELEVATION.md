# ELEVATION.md — THE FINAL JANG (Movement 2 · plan, 19-Jul-2026)

The last elevation. Rule of this plan: **every change is audit-proven necessary (RESEARCH.md §8 +
repo audit), class-level, at the choke point, with a regression gate from the exact artifact.**
Where the current build already meets the SOTA bar, the honest verdict is "no change" — rebuilding
for its own sake is the failure mode this app was built against (L5).

One truth stated once, because the constitution (I9) demands it: no software can *guarantee*
selection. What this plan buys is every controllable variable at state-of-the-art, proven error-free
on the owner-mode input space, and self-evolving so it stays that way. That is the whole of what
"ultra pro max" can honestly mean, and it is what we deliver.

   incorporated intelligently with zero re-instruction; no capability ever silently drops or caps
   another (cast-integrity is law).
3. **ZERO-ERROR IS EARNED, NOT PROMISED.** Enumerate the owner-mode input space; run the
   ≥1,000-case property harness (compile→sanitize→dedupe→cast-contract→one-page→PDF parse-back),
   all clean. He must be able to trust every résumé 100% without checking by hand.
4. **SELF-EVOLVING, WIRED, PERMANENT.** Judges read standards from the library; Pulse grows the
   library; vault-repair pushes every upgrade to HIS data; router swaps models by config; hunts
   derive from vision. Each loop PROVEN to fire on the default path — current in 3 months with
   zero edits from him.
5. **DISCOVERY = worldwide vision-ranked flood**, fresh, varied, demote-never-hide; a new
   vision-aligned role anywhere reaches him. Honest about what a personal app cannot replicate.
6. **MISSION-MANGAL / DEEPSEEK LAW.** Free-tier keys only, token-efficient, no waste, and NO
   tradeoff — not intelligence, speed, depth, or context. Where the naive path forces a tradeoff,
   design around it (routing, caching, compression, retry-not-degrade).

  all 11 trim levels × cast counts 1–6.

generate ≥1,000 vault×JD×op-sequence cases → compile → assert ALL of: no throw (sole documented

cast-contract + Last Judge declaration; one-page fit by the real Helvetica metrics; and for a
