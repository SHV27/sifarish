**I built my own hiring agent — and it refuses to lie.**

Everyone's shipping AI resume builders that fabricate. Recruiters know it — 91% now actively hunt fake
resumes. So I built the opposite.

SIFARISH runs on one law: **Compile truth. Draft everything. Send nothing.**

It keeps a ledger of what I can *actually* prove — shipped projects with live links, and an honest
"currently building" tier for the rest. Then for each role it **compiles** a resume from that ledger.
Not writes — *compiles*, like source code. A bullet with no evidence link? Compile error. It literally
cannot put a claim on my resume that I can't back.

Three things I'm proud of:

→ **The evidence is the interface.** Every tailored resume shows you which JD keywords made it (I have
proof), which are "currently building" (honest momentum), and which got *kept off* the resume entirely
because I have no evidence yet. You watch it refuse to lie, in real time.

→ **It's parse-back-tested.** It generates the PDF, extracts the text layer back out, and asserts the
ATS will read exactly what I wrote. Pretty two-column resumes fail this. Mine can't ship if it fails.

→ **It never sends.** No auto-apply, no LinkedIn bots (that's how people get restricted). It hands me
files + the official link. I click Send. Always me.

And the part I like most: it gets stronger on its own. When one of my "building" projects goes live on
GitHub, the app notices and offers to promote it — one click, and my resume compounds with real work.

Runs fully offline, zero API keys. Built solo as a five-seat studio in my own head: engineer, designer,
a referee that writes invariant tests, and a hostile recruiter that skims every output in 6 seconds.

Not looking for a resume that impresses. Looking for one that's *true* — and so well-armed that lying
would be a downgrade.
